# Política de Privacidad

## Introducción
Esta aplicación respeta tu privacidad y está diseñada para interactuar directamente con la API de AdGuard Private DNS sin recolectar datos personales por parte de los desarrolladores de la aplicación.

## Recolección de Datos
La aplicación no recolecta ni envía datos personales a servidores externos controlados por los desarrolladores.
Toda la comunicación se realiza directamente entre tu dispositivo y la API oficial de AdGuard DNS (`api.adguard-dns.io`).

## Credenciales
Tu API Key se almacena localmente en tu dispositivo de forma segura utilizando las capacidades de almacenamiento del sistema operativo (o base de datos cifrada local). No se comparte con terceros.

## Servicios de Terceros
La aplicación puede utilizar bibliotecas de terceros que podrían recolectar datos de uso anónimos o informes de fallos (por ejemplo, Sentry si está habilitado en la compilación), pero esto está sujeto a la configuración de compilación y consentimiento del usuario.

## Cambios en esta política
Nos reservamos el derecho de actualizar esta política de privacidad en cualquier momento. Se te notificará de cualquier cambio publicando la nueva política de privacidad en esta página.
